# Soft Reset Modem 

## Getting Started

##### 1. Manuel 

 
```
git clone https://github.com/gurkanakdeniz/soft-reset-modem-sample.git
cd soft-reset-modem-sample
npm install puppeteer
node restartModem.js
```

##### 2. Bash

```
git clone https://github.com/gurkanakdeniz/soft-reset-modem-sample.git
sh ~/soft-reset-modem-sample/modem_restart
```
